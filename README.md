Project Title
Pregnancy-Nutrition-Risk-Prediction-System-using-Machine-Learning

Domain :healthcare

Probelm statement:
Develope a machine learning-based application to predict pregnancy nutrition risk levels (Low, Medium, High) using patient health metrics.
The system helps in early identification of nutritional risks to support timely medical intervention.

System workflow
1. Data Collection
        ↓
2. Data Preprocessing
        ↓
3. Model Training
        ↓
4. Model Evaluation
        ↓
5. Model Selection (Best Algorithm)
        ↓
6. Model Deployment (FastAPI)
        ↓
7. User Interface (Django + Bootstrap)
        ↓
8. API Communication (Django → FastAPI)
        ↓
9. Prediction Generation
        ↓
10. Result Display (Risk Level + Visualization)

Benefits
Helps doctors identify nutritional risks early
Supports preventive healthcare decisions
Reduces chances of pregnancy complications
Improves maternal health monitoring


Tech Stack
Programming Language: Python
Data Processing: Pandas, NumPy
Machine Learning: Scikit-learn
Backend: FastAPI
Frontend: Django
UI Design: Bootstrap


Input Features
Age 
BMI 
Hemoglobin

Output Features
Low risk

High risks
